# AI Playbook — My AI Usage Handbook

[中文](README.md) | English

> A personal playbook for documenting and reusing how I work with AI.

## About

This repository is a collection of all my reusable AI collaboration assets — skills, tools, scripts, rules, roles, prompts, commands, along with thoughts and articles about using AI (stored in the `articles/` directory).

Everything is organized by function for easy search, reuse, and continuous iteration. Selected articles are converted to HTML and published via **GitHub Pages**.

## Directory Structure

```
ai-playbook/
├── articles/         # Thoughts, reflections & insights about AI usage
├── commands/         # Frequently used AI commands (quick shortcuts)
├── docs/             # Articles & GitHub Pages content
├── prompts/          # Prompt templates (reusable prompts)
├── roles/            # AI role personas (character & system prompts)
├── rules/            # Rule files (constraining & standardizing AI behavior)
├── scripts/          # Scripts (automation, batch processing, etc.)
├── skills/           # Skills (end-to-end workflows for specific scenarios)
├── tools/            # Tools (AI-assisted toolchain)
└── README.md         # This file
```

| Directory   | Description                                                                                        |
| ----------- | -------------------------------------------------------------------------------------------------- |
| `articles/` | Thoughts, reflections, and insights about AI usage and this repo — manually written or AI-assisted |
| `commands/` | Ready-to-use AI commands — "polish this text", "generate unit tests", etc.                         |
| `docs/`     | Articles about AI usage, tips, and HTML pages published via GitHub Pages                           |
| `prompts/`  | Proven, high-quality prompt templates ready to reuse or customize                                  |
| `roles/`    | AI role definitions — system-level personas and conversation rules for different contexts          |
| `rules/`    | Behavioral constraints for AI, ensuring output quality, style, and scope meet expectations         |
| `scripts/`  | Automation scripts that integrate AI into daily workflows                                          |
| `skills/`   | Complete skill bundles for specific tasks — prompt + rules + workflow                              |
| `tools/`    | Tools, configurations, and guides used alongside AI                                                |

## How to Use

1. **Browse**: Navigate by category to find the skill, prompt, or rule you need
2. **Reuse**: Copy prompts or rules directly into your AI conversations
3. **Learn**: Read articles in `docs/` for workflows and best practices
4. **Contribute**: Issues and PRs are welcome — share your AI tips!
