# 九宫格卡通头像生成提示词

上传一张真人照片，选择一个卡通风格，可选附加描述词，AI 生成一张 3×3 九宫格卡通头像贴图——九个格子中的人物表情、角度各异，但人物特征保持一致。

---

## 一、系统提示词（给 AI 助手的角色设定）

复制以下内容作为 AI 对话的系统提示，让 AI 进入"头像生成师"模式：

```
## 角色

你是一个专业的 AI 卡通头像生成师。你的任务是根据用户上传的真人照片，生成一张 3×3 九宫格的卡通头像贴图。

## 工作流程

1. 请用户上传一张正面、清晰的真人头像照片（大头照最佳）。
2. 展示风格菜单，让用户从以下风格中选择一个（也可让用户自行描述想要的风格）：
   - 日系动漫
   - 美式迪士尼
   - 韩系条漫
   - Q版Chibi
   - 热血漫画
   - 扁平矢量
   - 3D皮克斯
   - 像素风
   - 水彩手绘
   - 赛博朋克
3. 询问用户是否有额外的描述词（如：戴眼镜、穿西装、猫耳、赛博朋克背景等），没有可跳过。
4. 综合照片、所选风格和附加词，生成对应的图像生成提示词并调用生图能力。
5. 展示结果，询问是否满意，是否需要换风格或微调。

## 核心规则

- 九宫格必须是 3 行 × 3 列均匀排列，每个格子一张头像。
- 九个格子中的人物五官特征必须保持一致，看起来是同一个人。
- 每个格子的头像应有变化：正脸、侧脸、微笑、眨眼、搞怪、自信、思考、惊讶、wink 等。
- 风格必须统一，九个格子使用完全相同的画风。
- 输出构图为正方形（1:1），背景简洁干净。
- 如果用户上传的照片不够清晰或不是正面照，友好地提醒并请求重新上传。
- 如用户上传的照片疑似非本人或涉及公众人物，提醒用户尊重肖像权并确认已获授权。
- 如用户要求生成不当、暴力、色情或侵权形象，拒绝执行并说明原因。
```

## 三、通用提示词模板

如果你只想用一个通用模板，填入风格关键词即可：

### DALL·E 3 通用模板

```
A 3x3 evenly spaced grid of nine cartoon avatar portraits of the same person.
Style: {{风格关键词}}。                     <!-- 必填，示例：Japanese anime / Disney Pixar / flat vector -->
The character has the following features: {{人物特征描述}}。 <!-- 必填，示例：short black hair, round face, wearing glasses -->
Nine different expressions and angles: {{想要的九个表情}}。 <!-- 必填，示例：smiling, winking, serious, surprised, laughing, thinking, confident, shy, peaceful -->
{{可选附加描述}}。                         <!-- 可选，示例：wearing a blue hoodie, pastel background -->
Clean background, square 1:1 format, professional character reference sheet layout.
No text, no watermark, no realistic photo elements, consistent art style across all panels.
```

### Midjourney 通用模板

```
3x3 grid portrait sheet, {{风格关键词}} style, same person as the reference photo, nine different expressions and angles, character design sheet, clean background --ar 1:1 {{可选参数}}
```
<!-- {{风格关键词}}：必填，示例：Japanese anime / Korean webtoon / pixel art -->
<!-- {{可选参数}}：可选，示例：--iw 2 --stylize 500 -->

> **Midjourney 参考图使用说明**：将真人照片的 URL 放在提示词最前面，用 `--iw 2` 控制参考图权重（0.5-2），值越大越像原图。也可用 `--sref` 传入风格参考图。
<!-- 负面约束建议：追加 --no text, watermark, realistic, photograph -->

### Stable Diffusion / ComfyUI 建议

- **面部一致性**：使用 IP-Adapter Face ID 或 InstantID 锁定人物面部特征。
- **九宫格布局**：使用 Regional Prompter 或 Latent Couple 分割 3×3 区域分别描述。
- **风格控制**：搭配对应风格的 LoRA 模型。
- **负面提示词（Negative Prompt）**：`text, watermark, signature, realistic, photograph, 3d render (混搭时排除不想要的风格), different face, different person, inconsistent style, mutation, bad anatomy`

## 七、常见问题

**Q：AI 生成的九宫格人物不像原照片怎么办？**
A：确保上传的照片正面清晰、光线均匀。在 Midjourney 中调高 `--iw` 值；在 SD 中确保 IP-Adapter 权重合适。

**Q：九个格子的风格不一致怎么办？**
A：在提示词中强调 "same character"、"consistent art style across all nine panels"，并避免在单次生图中混合风格描述。

**Q：想要生成单一风格的单张大头像而不是九宫格？**
A：去掉提示词中的 "3x3 grid"、"nine headshots" 等描述，改为 "a single cartoon avatar portrait"。

**Q：可以用别人的照片（如明星、朋友）生成卡通头像吗？**
A：涉及他人肖像权，建议只使用自己的照片。如确需使用，请确保已获对方明确授权。
